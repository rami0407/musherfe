# Musherfe School Website

A dynamic website for Musherfe Elementary School, featuring an admin dashboard for content management.

## Features
- **Dynamic Content:** News, Initiatives, and Calendar events are fetched from Firebase.
- **Admin Dashboard:** Manage all content easily.
- **Responsive Design:** Works on all devices.

## Setup
1. Clone the repository.
2. Open `index.html` to view the site.
3. Open `admin.html` to access the dashboard.
